# Conway's Game of Life

## Интерактивная реализация клеточного автомата «Игры в жизнь» (Conway's Game of Life) на React и TypeScript

[![Deploy with Vercel](https://vercel.com/button)](https://game-of-life-zeta-one.vercel.app/)

Стек: React, TypeScript, Vite, CSS Modules.

«Игра в жизнь» — клеточный автомат, созданный Джоном Конвеем. Следующее состояние поля вычисляется по простым правилам, что приводит к вызову сложных и интересных паттернов. Этот проект — практическая реализация алгоритма на современном стеке фронтенд-разработки.
Функционал

    Симуляция: Точное следование правилам Конвея.

    Управление: Старт / Пауза / Рисование.

    Взаимодействие: Можно добавлять новые клетки.

    Генерация: Создание поля со случайным начальным состоянием.

    Адаптивность: Корректное отображение на разных устройствах.

Как запустить локально

    Клонируйте репозиторий:
    bash

    git clone https://github.com/mashaserova/gameOfLife.git

    Перейдите в папку проекта:
    bash

    cd gameOfLife

    Установите зависимости:
    bash

    npm install

    Запустите dev-сервер:
    bash

    npm run dev

    Откройте браузер по адресу http://localhost:5173.
