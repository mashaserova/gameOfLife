# Conway's Game of Life

## Интерактивная реализация клеточного автомата «Игры в жизнь» (Conway's Game of Life) на React и TypeScript

Стек: React, TypeScript, Vite, CSS Modules.
О проекте

«Игра в жизнь» — клеточный автомат, созданный Джоном Конвеем. Следующее состояние поля вычисляется по простым правилам, что приводит к emergence сложных и интересных паттернов. Этот проект — практическая реализация алгоритма на современном стеке фронтенд-разработки.
Функционал

    Симуляция: Точное следование правилам Конвея.

    Управление: Старт / Пауза / Шаг вперёд / Сброс.

    Взаимодействие: Клик по клетке меняет её состояние. Можно рисовать паттерны.

    Генерация: Создание поля со случайным начальным состоянием.

    Адаптивность: Корректное отображение на разных устройствах.

Как запустить локально

    Клонируйте репозиторий:
    bash

    git clone https://github.com/mashaserova/gameOfLife.git

    Перейдите в папку проекта:
    bash

    cd gameOfLife

    Установите зависимости:
    bash

    npm install

    Запустите dev-сервер:
    bash

    npm run dev

    Откройте браузер по адресу http://localhost:5173.
